<h1>Natural Language Data Visualization with LLM Workflows</h1>
You can easily upload dataset, then ask the llm to generate a chart from the data.
<br>
What it do:
    1. Upload dataset
    2. Prompt a chart
    3. Approve or prompt an edit
    4. Preview chart and code
